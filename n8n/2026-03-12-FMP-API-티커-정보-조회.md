### Context

Financial Modeling Prep(FMP) API 를 활용하여 특정 주식 티커(예: TSLA) 의 실시간 시세 정보를 가져오려는 상황입니다. FMP 는 다양한 금융 데이터를 제공하며, 특히 `quote` 엔드포인트를 통해 주식의 현재 가격, 변동폭, 거래량 등 약 17 가지의 상세 정보를 JSON 형식으로 반환합니다. 최근 API 엔드포인트가 `stable` 버전으로 변경되었으며, 인증 방식은 쿼리 파라미터(Query Parameter) 를 사용하는 것이 간편합니다.

### Core

FMP API 에서 티커 정보를 조회하기 위해서는 `symbol` 파라미터와 `apikey` 파라미터를 URL 에 포함해야 합니다. 아래는 파이썬과 n8n 에서 적용하는 구체적인 설정 방법입니다.

**1. API 엔드포인트 및 파라미터 구조**

*   **Base URL**: `https://financialmodelingprep.com/stable/quote`
*   **필수 파라미터**:
    *   `symbol`: 조회할 주식 티커 (예: `TSLA`, `AAPL`)
    *   `apikey`: FMP 가입 시 발급받은 API 키

**2. 파이썬 예제 코드**

```python
import requests

# 설정
api_key = 'YOUR_API_KEY'  # 가입 시 발급받은 키
symbol = 'TSLA'

# URL 구성
url = f"https://financialmodelingprep.com/stable/quote?symbol={symbol}&apikey={api_key}"

# 요청 및 응답
response = requests.get(url)
data = response.json()

# 결과 출력
if data:
    print(f"티커: {data[0]['symbol']}")
    print(f"현재가: {data[0]['price']}")
    print(f"변동폭: {data[0]['changesPercentage']}%")
else:
    print("데이터를 가져오지 못했습니다.")
```

**3. n8n HTTP Request 노드 설정**

n8n 에서 FMP API 를 호출할 때는 `HTTP Request` 노드를 사용합니다.

*   **Method**: `GET`
*   **URL**: `https://financialmodelingprep.com/stable/quote`
*   **Query Parameters**:
    *   **Name**: `symbol` | **Value**: `TSLA` (또는 변수)
    *   **Name**: `apikey` | **Value**: `YOUR_API_KEY`

> **주의**: `apikey` 파라미터 이름은 반드시 소문자로 정확히 입력해야 합니다. 대소문자 구분 (`ApiKey`, `APIKEY` 등) 이 적용되어 인증 실패가 발생할 수 있습니다.

### Insight

FMP 의 `quote` 엔드포인트는 요청한 티커에 대해 약 17 가지의 상세 정보를 반환합니다. 주요 필드는 다음과 같습니다.

*   `symbol`: 티커 심볼
*   `name`: 회사명
*   `price`: 현재 주가
*   `changesPercentage`: 전일 대비 변동률 (%)
*   `change`: 전일 대비 변동액
*   `dayLow`, `dayHigh`: 당일 최저/최고가
*   `yearLow`, `yearHigh`: 52 주 최저/최고가
*   `marketCap`: 시가총액
*   `volume`: 거래량

이 엔드포인트는 `stable` 버전을 사용하므로 URL 구조가 비교적 안정적이지만, API 키 관리 시에는 환경 변수를 활용하여 보안에 유의해야 합니다. 또한, 무료 플랜의 경우 분당 요청 횟수 제한이 있으므로 대량 데이터 수집 시 주의가 필요합니다.
